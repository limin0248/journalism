# 财经记者炼成记

基于 Boss 直聘 300 条招聘数据的财经记者职业指南网站。

## 内容章节

1. **开始** — 财经媒体发展历程与研究方法
2. **职位** — 职业定义、条线分布、招聘单位与薪资待遇
3. **市场** — 学历要求、地域分布与技能网络
4. **经验** — 前辈经验总结

## 技术栈

- HTML5 + CSS3（原生，无框架）
- 响应式布局，适配移动端
- 原生 JavaScript 交互

## 部署

通过 GitHub Pages 部署，自定义域名 `sensible.cyou`。

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。
